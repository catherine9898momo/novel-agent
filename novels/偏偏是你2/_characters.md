# 人物设定

## 女主：沈鸾

**身份**：镇北将军府嫡女，自幼随父兄习武

**外貌**：
- 明艳如玫瑰，眉眼间带着几分骄矜
- 喜着红衣，骑马射箭时英姿飒爽
- 笑起来明艳动人，恼起来也明艳动人

**性格**：
- 热烈、娇蛮，却娇蛮得可爱
- 骄傲但不刻薄，嘴硬心软
- 像一朵带刺的玫瑰，刺是保护，也是锋芒
- 有话直说，不会藏着掖着，喜怒形于色
- 看起来任性，实则有自己的底线和坚持
- 关键时刻有决断力，不是只会反应的人，会主动出击

**武功**：
- 自幼习武，剑法、骑射皆通
- 武功水平：中上，日常能自保，但遇到当世高手无法对抗
- 擅长：快剑、骑射
- 弱点：内力不足，持久战吃亏

**背景**：
- 父亲是镇北将军，镇守北疆十余年
- 母亲在她三岁时病逝，由父兄抚养长大
- 从小在军营长大，见惯生死，性格爽利
- 对母亲的记忆极少，只知道母亲是京城官宦人家的女儿，不适应北疆生活
- 此次入京，父亲说是为她议亲，但沈鸾隐约觉得事情没那么简单

**不知道的秘密**：
- 母亲的死不仅是因为水土不服，更是因为二十年前"靖安之变"中参与的一个惊天秘密——狸猫换太子
- 父亲召她入京议亲的真正原因，是察觉到有人在追查旧事，想把她尽快嫁到安全的人家

**内心渴望**：
- 表面：想要自由，不想被束缚在深宅大院
- 深层：想要被理解，被看见真实的自己，而不是"将军府嫡女"的标签
- 最深处：想要知道母亲到底是怎么死的，父亲到底在隐瞒什么

**人物弧光**：
- 从用骄蛮保护自己 → 在与裴晏的相处中学会真正敞开 → 发现父亲的秘密后世界观崩塌 → 独自去调查真相、做出判断 → 最终主动出手为裴晏争取自由，完成从被保护者到守护者的蜕变

## 男主：裴晏

**身份**：当朝皇帝第三子，封静王。实际上是前朝太子遗孤，出生时被沈恪夫妇偷换，冒充皇帝之子长大。

**外貌**：
- 清俊冷峻，眉目如画却总带着疏离
- 常着玄色或素色衣袍，气质孤傲
- 左臂有一道自出生起就有的伤疤，宫中记载为难产所致，实为靖安之变中流矢所伤

**性格**：
- 外表冷漠，内心有裂缝
- 孤傲、偏执、深情，对世俗不屑
- 不善言辞，用行动而非语言表达
- 对亲近之人极尽温柔，对旁人冷若冰霜
- 有自己的是非观，不盲从权威
- 从小有一种说不清的"不属于这里"的感觉，但从未怀疑过自己的身份

**武功**：
- 当世顶尖高手，剑法出神入化
- 擅长：剑、轻功
- 左臂旧伤是弱点，阴雨天会痛，但极少人知道

**背景**：
- 真实身份：前朝太子之子。靖安之变中，前朝太子被沈恪所斩，太子妃产后力竭而亡
- 出生时被沈恪夫妇与皇帝侧妃所生的女婴对换，以皇帝第三子的身份长大
- 皇帝将他当亲生儿子养大，封静王，但父子关系始终有一种微妙的疏离——裴晏将此归因于帝王天性多疑
- 朝堂之上，他能力出众，但因性格孤傲不结党，显得形单影只
- 他完全不知道自己的真实身世

**内心渴望**：
- 表面：想要弄清楚父皇对自己若即若离的原因，想要证明自己
- 深层：想要一个真正属于自己的归属，一个不会因身份而改变的联结
- 他一直有一种模糊的"错位感"，觉得自己与皇家格格不入，但不知道为什么

**人物弧光**：
- 从孤傲自持 → 被沈鸾的真诚打动，开始敞开 → 发现自己的身世是一个谎言，整个存在的根基被抽走 → 在"我是谁"的追问中挣扎 → 最终选择：身份是假的，但感情是真的，用沉默守护沈鸾一家的安全

## 配角

### 沈恪（女主父）
- 镇北将军，忠勇刚直，但背负着二十年前的惊天秘密
- 靖安之变中亲手斩杀前朝太子，之后不忍杀害襁褓中的婴儿，与妻子合谋将婴儿与皇帝侧妃之女对换
- 对女儿宠爱有加，但不擅表达
- 与皇帝有旧交，是少数敢直言进谏的臣子
- 这些年镇守北疆，一半是职责，一半是远离京城是非
- 对裴晏的态度极其复杂：愧疚（杀了他父亲）、恐惧（怕真相暴露）、隐秘的关注（毕竟是自己救下的孩子）
- 急于为沈鸾议亲的真正原因：察觉柳贵妃在追查旧事，想把女儿嫁到与此事无关的安全人家，万一事发，女儿至少有退路

### 沈铮（女主兄）
- 沈家长子，性格沉稳
- 妹控，对沈鸾百般维护
- 武功高强，是父亲的左膀右臂
- 不知道靖安之变的秘密（沈恪没有告诉他），但隐约察觉父亲有心事

### 沈夫人（女主母，已故）
- 京城官宦人家的女儿，知书达理，外柔内刚
- 靖安之变中随丈夫在侧，亲手完成了换婴之事
- 随丈夫赴北疆后，一方面不适应苦寒气候，另一方面长年活在秘密败露的恐惧中
- 在沈鸾三岁时病逝，表面是水土不服体弱多病，实际上是长年精神重压与愧疚（被换走的女婴未能活过北疆的第一个冬天）双重摧垮
- 她的死是沈恪心中最深的痛——妻子不只是被北疆的苦寒害死的，更是被这个秘密压死的

### 皇帝
- 深沉多疑，但并非无情
- 完全不知道裴晏不是自己亲生——他将裴晏当亲儿子养大
- 对裴晏既欣赏又不安：裴晏太出色、太孤傲、太不像自己的任何一个儿子。这种"不像"让他隐隐不安，但他将之归因于裴晏像其"生母"（那位侧妃）
- 当年那位侧妃在"产后"不久便郁郁寡欢（因为真正的女儿被换走了），后来出家为尼，皇帝对此一直心怀愧疚
- 储位之争中，他在裴晏和柳贵妃之子之间摇摆不定

### 柳贵妃
- 出身名门柳家，心机深沉
- 柳家在靖安之变中是关键的从龙势力，家族手中握有大量当年的隐秘
- 柳贵妃通过家族旧人和宫中眼线，查到了二十年前换婴之事的蛛丝马迹
- 她不敢贸然告发——如果证据不足，皇帝会认为她构陷皇子，她自己反而死路一条
- 所以她在暗中搜集铁证：沈家是最关键的线索。沈鸾入京，对她来说是绝佳的机会
- 她有一个亲生皇子（四皇子），正在争储位。裴晏是最大的竞争对手
- 她的终极目标：用铁证在合适的时机揭露裴晏身份，一举扳倒裴晏，送自己的儿子上位
- 她不是脸谱化的坏人——她是一个为了儿子前途不择手段的母亲

### 顾清婉
- 京城贵女，外表温婉，内心骄傲
- 对裴晏有好感，视沈鸾为对手
- 不是反派，只是立场不同
- 后期主动退出，两人成为朋友

