# 《烟雨长安》故事摘要（截至第4章）

## 已发生的关键事件

**第1章：入京**
- 沈晚棠在雨夜回到长安，七年未归
- 她被家族召回是为了替嫁——原定新娘沈晚雁因天花毁容
- 马车中她意识到自己只是替补品，是沈家的工具
- 暗处有神秘人监视她的归来

**第2章：初见**
- 沈晚棠回到沈府，见到父亲沈云骞和母亲林氏
- 父亲直言不讳告知她要替姐姐嫁给镇北侯谢危楼
- 林氏虽有愧疚，但不敢违抗丈夫
- 沈晚棠在府中与谢危楼擦肩而过，感受到他的冷漠和危险
- 她意识到这桩婚事是政治联姻，自己毫无选择余地

**第3章：宫宴**
- 皇帝设宴公开宣布赐婚
- 沈晚棠与谢危楼首次正式对话，两人交杯时谢危楼说"你我都没得选"
- 宴席上太子一党的李昀暗讽谢危楼，被谢危楼冷酷回击
- 谢危楼试探沈晚棠在江南七年的真实情况，表示已在调查
- 沈晚棠察觉谢危楼比想象中更危险，会把所有事查个底朝天
- 宴后马车中，沈晚棠意识到自己的祸福已与谢危楼绑在一起

**第4章：姐妹**
- 深夜，沈晚棠去西院见被毁容的姐姐沈晚雁
- 沈晚雁摘下面纱，露出满是疤痕的脸，对妹妹充满怨恨和嫉妒
- 她质问"凭什么是你"，认为沈晚棠抢走了本该属于她的婚事
- 沈晚棠回忆起姐妹往事，曾经沈晚雁救过她的命
- 沈晚棠询问天花来源，发现去年春天太子府的人来过，太子还赏赐过胭脂
- 她怀疑沈晚雁的天花来得蹊跷，可能与太子有关
- 沈晚雁最后说"你会后悔的"，像一句诅咒
- 沈晚棠意识到姐妹情分已碎，两人站在了对立面

## 人物状态变化

**沈晚棠**
- 从被动接受命运到开始主动观察和思考
- 察觉到谢危楼的危险性和洞察力
- 发现姐姐心理已扭曲，姐妹关系破裂
- 怀疑天花事件背后有太子的阴谋

**谢危楼**
- 对沈晚棠保持警惕和试探
- 已开始调查她在江南七年的真实情况
- 在宫宴上展现冷酷手段

**沈晚雁**
- 因毁容和失去婚约，心理严重扭曲
- 对沈晚棠充满嫉妒和怨恨
- 暗示将有报复行动

## 未解决的伏笔

1. **沈晚棠在江南七年做了什么？** 谢危楼正在调查，她似乎有不能让人知道的秘密
2. **天花的真正来源** 太子府的人去年春天来过，太子赏赐过胭脂，时间太过巧合
3. **沈晚雁的报复** 她说"你会后悔的"，预示着将采取行动
4. **沈家的真实立场** 父亲让沈晚棠嫁给谢危楼，是否有监视的目的？
5. **暗中监视者** 第1章出现的神秘人是谁派来的？
6. **太子的图谋** 他为何要毁掉沈晚雁？对谢危楼又有什么计划？

## 情感线进展

- 沈晚棠与谢危楼：互相试探阶段，建立了"做戏"的默契，但彼此防备
- 沈晚棠与沈家：从依附到开始质疑，感受到被当作工具的痛苦
- 沈晚棠与沈晚雁：姐妹情分彻底破裂，从亲情变为对立
