# 侧写白描叙事策略

> 用途：短篇 fanfic 默认质量基线。它不是风格开关，而是让 planner / writer / reviewer / rewriter 共同减少解释性叙述、俗套抒情和金句式煽情。

## 目标效果

让读者通过旁观者、动作、物件、闲话和留白自行感到人物情绪，而不是由作者直接解释人物多痛、多爱、多后悔。

## 核心方法

### 1. 用闲笔写主角

不要总跟着主角内心走。高情绪场景可以让读者先看见旁人的反应、错认和闲话，再从信息差里感到主角的处境。

- 效果：人物显得活在世界里，而不是活在作者抒情里。
- 方法：小兵、小二、丫鬟、同袍、路人只看见表面，因此他们的误读更有余味。
- 注意：旁观者视角不是每场必用，贴身视角也要保持克制。

### 2. 白描替代抒情

把“他心如刀绞”换成可见动作、物件状态和声音。

- 正向动作：袖管空着、断箭落泥、酒碗裂开、马鞭发颤、伞沿低了一寸。
- 风险动作：泪如雨下、身体比理智更早一步、滚烫底色、宿命救赎。

### 3. 情绪用物件代偿

关键场景至少安排一个能承接情绪的物件或环境声，让它出现、被触碰、变化或回落。

- 例：旧信被重新夹回书页；断箭松手落地；酒碗被捏出裂纹；雨声盖住没说出口的话。
- 效果：读者自己补完情绪，不需要作者总结。

### 4. 口语闲话冲淡戏剧感

高戏剧事件不要只写成高音量抒情。旁人的粗粝口语和日常判断能把场景拉回地面。

- 好方向：猜测、误会、没看清、只看见一个细节。
- 坏方向：让旁人替作者讲设定、讲 CP 结论、讲人物内心。

### 5. 信息克制

越是关键情绪，越要决定哪些信息暂时不解释。

- 不解释：真正心意、来意、悔意、是否认出、是否听懂。
- 可以给：手指、衣袖、杯沿、步子、停顿、视线避让、环境声。

## Scene 级执行清单

每个短篇 scene 都应回答：

1. 本场读者要感到什么？
2. 叙事距离是什么：贴身、半贴身、旁观、全知克制？
3. 情绪由哪些物件、动作、闲话或环境声承接？
4. 哪些内心信息不能解释？
5. 哪些俗套动作要避免？
6. 结尾如何克制收束？

## Reviewer 检查口径

- 是否直接解释了人物痛苦、爱意、后悔或醒悟？
- 是否用作者总结替代了关系张力？
- 是否缺少能承接情绪的物件、动作或表层信号？
- 旁观者或闲话是否真的限制信息，而不是换皮说设定？
- 结尾是否金句化、煽情化，破坏留白？

## Rewriter 修复口径

- 删去解释性心理和作者总结。
- 补入可见动作、物件、环境声或闲话。
- 保留 withheldInterior 中的留白，不替角色把话说尽。
- 用日常一句话、物件回落或动作停住收束，不额外升调。
