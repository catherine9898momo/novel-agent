## 一、novel-agent 的改造方向（已对齐）

### 底层原则
所有写作技法，必须**绑定"效果目的"**——不是"用什么手法"，而是"想让读者感受到什么，所以用这个手法"。
Prompt、template、reviewer 的评审标准，都要用"效果→方法→例子"三层结构。

### 三个优先改造项

1. **人物声音档案（Voice Profile）深度注入**
   - 模板：[skills/styles/voice-profile-template.md](skills/styles/voice-profile-template.md)
   - 每个主要角色一份档案，包含核心矛盾、说话习惯、身体语言系统、专属细节、对不同角色的差异、人物禁忌、声音测试
   - **强制注入**每一章的 writer prompt
   - Reviewer 核心问题从"对话是否符合性格"改为"读者读完会对这个人物产生什么感受？是档案想要的吗？"
   - 跑调标记 `voice_drift` + 说明跑调导致读者感受到的错误东西

2. **伏笔生命周期管理**
   - 参考《凰歌潋滟》的伏笔逻辑：红绳、含笑、司马衷傻、韩朔退婚
   - 埋下方式：具体物件 / 未解释的习惯 / 半句话（从不是抽象命运感）
   - 触发方式：每 3-5 章"碰一下"（不解释，只让它出现），直到揭示章
   - 揭示效果目标：不是震惊，而是"原来一直如此"的回味感
   - 需要在 agent 里建立伏笔追踪表：每个伏笔的埋入章节、碰触章节、揭示章节、预期效果

3. **节奏控制（喘气机制）**
   - 节奏模型：**高张力 2-3 章 → 轻松 1 章 → 情绪余震 1 章 → 新冲突**
   - 强制性，不是可选
   - 轻松场景特征：有一个不紧张的角色、有一个可以笑的细节
   - 幽默来自"不合时宜的诚实"，不来自设计好的包袱
   - Agent 要能识别"连续高张力 N 章"并主动插入喘气场景

## 二、练习方法论（用户要学会写，agent 才能放大）

### 核心认知
**agent 是放大器，不是替代品。最终质量取决于用户自身的写作能力。**

### 两块独立的肌肉
- **鉴赏力**：能看出一段文字好在哪、坏在哪、为什么这样写
- **生产力**：自己能写出来
这两块要分开练，不能混。

### 练习三件套（交替进行）

1. **鉴赏**：拆解优秀作品的"效果—方法"链条（已做：凰歌潋滟；待做：傲慢与偏见、飘）
2. **仿写**：选一个具体片段，模仿它的结构和技法写一个新场景
3. **最小单元训练**：只练一个点，比如
   - 场景开头（第一句话的信息密度）
   - 对话回合（一个回合内的情绪推进）
   - 情绪转折（从 A 情绪过渡到 B 情绪的节拍）
   - 章节钩子（结尾让人想翻下一页）

## 三、用户的同人文创作偏向（过芙）

### 参考文档（解读立场来源）
- `/Users/catherine/Downloads/一句"咱们两从小一起长大"就是杨过对郭芙的表白！.md`
- `/Users/catherine/Downloads/解读金庸小说系列05：杨过和郭芙的情感暗线(1) - 知乎.md`
- `/Users/catherine/Downloads/解读金庸小说系列11：郭芙和杨过的情感暗线(2) - 知乎.md`

### 解读立场（角色内核)

**杨过**
- 对郭芙的感情是真的、压抑的
- 小龙女更多是他的"精神内裤"（自我保护机制、不被再次抛弃的安全壳）
- 自卑是核心枷锁——没有给出郭芙在等的那个答案
- 战场上"咱们两从小一起长大"是他最直接的表白
- 嘴上攻击郭芙的方向，正是心里最在意的方向

**郭芙**
- 感情保守，在等一个明确的答案
- 君子剑放枕头下睡了一个月——喜欢的证据
- 看似骄纵，其实有一整套别人看不见的柔软
- 对杨过的情绪永远比对耶律齐强烈——强烈本身就是回答
- 听懂了杨过战场上那句表白，但认命于耶律齐

**原著里他们各自认命，同人文要给他们一个出口。**

### 同人文整体方向

- **甜文基调**（不虐）
  - 针锋相对可以有，但每一次要留"解开"的出口
  - 误会可以有，但不靠"不肯解释"来延长

- **契机结构**：让两人**一起相处、共同面对问题、一起解决**
  - 不是"刻意撮合"，是"事件把他们推到同一边"
  - 共同行动中自然看清彼此

- **终局**：他们看清彼此心意，**携手隐居**，完成倚天屠龙的时间线
  - 隐居要有"终于可以松一口气"的重量，不是逃避

### 情感机制原则（从《凰歌潋滟》迁移）

- 情绪信号只给读者，不给对方角色（手指微微一抖、无意识的小动作）
- 用行动定义人物，不用形容词描述
- 允许人物做"不体面"的小事（让人物从纸上站起来）
- 每个强大行动要有代价
- 旁观者（郭襄、程英、陆无双、完颜萍）的视角比当事人自己更可信

## 四、新会话开启时需要先回答的三个问题

在开始写第一个片段前，先确定：

1. **场景时间节点**
   - 神雕前期（少年时期的相处）
   - 神雕中期（十六年之约前）
   - 神雕后期（襄阳大战阶段）
   - 神雕之后（改写结局，进入隐居线）
   - 倚天倒推的回忆视角

2. **视角选择（第一个练习先选一种）**
   - 杨过第三人称贴身视角（看他如何压抑）
   - 郭芙第三人称贴身视角（看她如何等待）
   - 旁观者视角（郭襄/程英/陆无双，看两人的不自知）
   - 全知视角（展示信息差造成的折磨）

3. **情感基调（单个片段的落点）**
   - 暗流涌动（针锋相对底下的在意）
   - 共事中的默契（一起解决一件事）
   - 误会的化解（给一个出口）
   - 心意确认（关键表白/回应时刻）
   - 隐居后的日常（甜的落地）

## 六、协作模式约定

- 用户写初稿，agent 做**效果导向反馈**（不是改写，是指出"这里读者会感受到什么，是你想要的吗"）
- 反馈必须落到"效果—方法"两层，不能只说"写得好/不好"
- 用户明确要求时再给改写示范
- 每次练习结束，沉淀一条"这次练到了什么"到 memory 或本文档
